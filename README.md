# どこでもナウル

ナウル共和国の**実寸シルエット**を地図上に重ねて、大きさの感覚を体感するWebアプリです。  
正式版 v1.0.0 です。

**公開ページ:** [https://issonokoto.github.io/dokodemo-nauru/](https://issonokoto.github.io/dokodemo-nauru/)

地図を動かすと、画面中央にナウル（またはナウルくん）がついてきます。赤道付近では小さく、高緯度ではメルカトル図法の影響で大きく見えます。

## できること

- 国の境界 / ナウルくん の切り替え
- 色・回転・反転
- 比較スポット（東京・大阪・博多など）と緯度経度ジャンプ
- 現在または1995年時点の自治体、国・地域199件、島303件、湖・池254件を検索して、ナウルと面積・輪郭を比較
- マイ地点（最大2件）
- スクリーンショット保存と X / Instagram / LINE への共有導線

- 市区町村・都道府県の面積: 国土地理院「全国都道府県市区町村別面積調」（令和8年4月1日）

- 歯舞群島の面積: e-Stat「統計で用いる標準地域コード」用語の解説

- 国・地域の輪郭: [Natural Earth 1:10m Cultural Vectors](https://www.naturalearthdata.com/) / [geoBoundaries gbOpen](https://www.geoboundaries.org/)
- 島・湖／池の輪郭: © [OpenStreetMap](https://www.openstreetmap.org/copyright) contributors / [Natural Earth 1:10m Physical・Cultural Vectors](https://www.naturalearthdata.com/)（加工して表示）

- メルカトル図法上の実寸比較という体験は、既存の True Size 系・「どこでも〜マップ」系の着想を参考にしています

## 注意

本サイトはスケール感を体験するシミュレーターです。現在の市区町村・都道府県と、島・湖／池の大部分の面積には国土地理院の公表値を使用しています。1995年時点の自治体と公表値を紐づけられない一部の島・湖／池は、境界データから算出した概算です。島は国土地理院の1km²以上一覧を基準にし、竹島と北方四島を追加収録しています。湖・池はWikipedia「日本の湖沼一覧」に掲載された各都道府県の湖沼を中心に、輪郭を取得できたものを収録しています。係争地域は日本政府の呼称・区分に基づき、現在の実効支配についても比較画面に明記しています。シルエットの見た目やメルカトル図法上の表示は、厳密な測量には使用できません。

島・湖／池の輪郭は対象ごとの高精細データに分割し、検索時には一覧だけ、移動・比較時には選択した1件だけを読み込みます。

海外の島は77件を収録しています。ハワイ主要8島、フィリピン・インドネシア周辺の主要島に加え、グリーンランド、グレートブリテン島、マダガスカル島などの世界最大級の島と、イースター島、マンハッタン島、アルカトラズ島など面積によらず著名な島も検索できます。同名の硫黄島は東京都と鹿児島県を分けて検索できます。

海外の湖・池は26件を収録しています。カスピ海、五大湖、ビクトリア湖、バイカル湖などの世界最大級の湖に加え、死海、レマン湖、コモ湖、ネス湖、ブレッド湖、ウォールデン池など、観光・歴史・文化面で広く知られた水域も収録しています。

国・地域は台湾、北朝鮮、パレスチナ、コソボ、クック諸島、ニウエを含む199件を収録しています。国境表現は Natural Earth の日本向け表示を基準とし、極小国・小島嶼国と台湾は geoBoundaries の高精細輪郭を使用しています。

共有ボタンは、ブラウザの制限により画像の自動添付はできません。保存 → クリップボードへコピー → 投稿画面で貼り付け、の流れになります。
